이 폴더는 제가 읽었던 논문 중, 중요하거나 대표적인 논문의 공식 코드/참고 코드를 기반으로, 모델과 구현 내용을 직접 이해하고 정리하기 위해 구성되었습니다.

### 진행 방식

1. **코드 주석 추가**
   - 논문에서 제안하는 알고리즘과 구현 아이디어를 이해하기 위해,
     원본 코드 곳곳에 한글 주석을 추가합니다.

2. **폴더별 README 작성**
   - 각 논문의 상위 폴더에 README.md 파일을 작성합니다.
   - 논문 별로 다음 내용을 정리합니다.

     - 논문의 대표적인 아이디어
     - 코드 구현 방식
     - 모듈 간의 연결 방법
